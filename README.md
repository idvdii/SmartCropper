# 📸 SmartCropper (Dataset Expert Tool)

**SmartCropper** 是一款专为深度学习（LoRA/SDXL）数据集预处理设计的高效图片裁剪工具。它解决了传统修图软件在制作训练集时效率低下、容易产生黑边、分辨率不统一的痛点。

> **核心使命**：让数据准备工作从“痛苦的体力活”变成“解压的点击游戏”。

## ✨ 核心亮点 (Key Features)

- **🚫 拒绝黑边**：智能边界吸附算法，无论如何拖拽缩放，永远不会裁出黑边，保障训练集纯净。
- **🎯 锁定输出**：支持强制锁定分辨率（如 `512x512`, `1024x1024`），软件会自动进行高质量缩放 (Lanczos)，无需后期二次处理。
- **🤖 AI 画质增强**：内置 **4x-UltraSharp** 模型。当你裁剪小图并放大时，AI 会自动介入修复画质，拒绝模糊。
- **🚀 极速流操作**：
    - **视口优化技术**：8K 超大图片也能跑满 60 帧丝滑拖拽。
    - **全键盘流**：`空格`保存下一张，`Del`丢弃，`Alt+方向键`微调选框。
- **♻️ 影子回收站**：误删原图？没关系，原图和对应的结果图都会进入专用回收站，随时一键恢复。

## 🛠️ 安装与运行 (Installation)

请前往 [Releases 页面](https://github.com/idvdii/SmartCropper/releases) 下载最新的 `.zip` 压缩包，解压即用。

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   # 如果下载慢，请使用清华源：
   # pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```
2. **启动程序**
   ```bash
   python run.py
   ```
   或者双击`启动工具.bat`文件启动
## 🎮 快速上手指南
1. **将原始图片放入 *set_image* 文件夹。**
2. **运行软件，在顶部下拉框选择工作目录。**
3. **选择合适的宽高比 / 自定义。**
4. **操作：**
  -  左键拖拽：移动图片    
  -  滚轮：缩放图片    
  -  空格 / →：保存并切换下一张    
  -  Delete：丢弃当前图片
